# ClubHub - Community Club Management System

## Person 2: Discovery Feed & Club Profile

This is the implementation for Person 2's tasks in the Club Management System project.

### Features Implemented

#### Discovery Feed (index.html)
- Display all clubs with their information
- Filter clubs by category
- Search clubs by name, category, or description
- Show upcoming events

#### Club Profile (club.html)
- Display club information (name, category, description)
- Show member count (seed members + current user if joined)
- Join/Leave club functionality
- Display club members
- Show club-specific upcoming events
